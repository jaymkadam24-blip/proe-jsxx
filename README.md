# ProE — Complete IT & Apple Solutions

A modern web platform for **Prowesses Enterprises**, built with React, Vite, and Supabase. The site showcases IT services, Apple device repair, enterprise infrastructure, and managed networking — and includes a secure contact form powered by a Supabase Edge Function.

---

## Tech Stack

- **React** — Frontend UI
- **Vite** — Build tool and dev server
- **React Router** — Client-side routing
- **Tailwind CSS** — Styling
- **Supabase** — Database + Edge Functions (backend)
- **Netlify** — Hosting and deployment

---

## Getting Started

### Prerequisites

- Node.js v18+
- npm
- A Supabase account and project
- A Netlify account (for deployment)
